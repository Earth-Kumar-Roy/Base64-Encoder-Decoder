# Base64 Encoder & Decoder

A lightweight web tool for encoding and decoding Base64 data. It supports both text and file operations, provides instant copy/download actions, and includes a clean, responsive interface.

## Features
- Encode text or files to Base64  
- Decode Base64 to text or downloadable files  
- Copy-to-clipboard support  
- Auto-file download for decoded data  
- Responsive, modern UI with tab navigation

## Usage
1. Open the HTML file (`index.html`) in any modern browser.  
2. Select **Encoder** or **Decoder**.  
3. Choose **Text** or **File** mode, then perform the desired action (Encode / Decode).

## Notes
- Maximum file size for encoding: **10 MB**  
- Decoded files download as `.txt` unless adjusted manually  
- Ensure Base64 input is valid to avoid decoding errors
